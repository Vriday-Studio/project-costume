This sample shows the use of shadergraph for AR material.

Universal Render Pipeline overview: https://docs.unity3d.com/Packages/com.unity.render-pipelines.universal@14.0/manual/index.html
About Shader Graph: https://docs.unity3d.com/Packages/com.unity.shadergraph@14.0/manual/index.html

For Shader Graph sample install Universal RP (10.4.0+) and Shadergraph (10.4.0+) via package manager
And use UniversalRenderPipelineAsset in the field "Project Settings\Graphics\Scriptable Render Pipeline Settings"
